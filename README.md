# Kindergarten Application Tracker

Static kindergarten application tracker for Fall 2027 NYC applications.

## Publish with GitHub Pages

1. Create a new public GitHub repository, for example `kindergarten-application-tracker`.
2. Upload `index.html`, `.nojekyll`, and this `README.md`.
3. In GitHub, go to Settings -> Pages.
4. Under "Build and deployment", choose "Deploy from a branch".
5. Select the `main` branch and `/ (root)`.
6. Save. GitHub will show the public website URL after it deploys.

The tracker saves checkbox and note data in each viewer's browser local storage. Other people will see the same page, but they will not see your private local notes.
